项目组不可能只有一个人写全部代码，基本流程如下

- 项目经理/架构师将先搭好的框架上传 ==Push== 到 Github 上

- 然后开发人员克隆 ==Clone== 到自己电脑上

- 等开发人员进行开发，开发完成的代码需要推送 ==Push== 到 Github 上，==需要权限==，
    - 假设推送不需要权限的话，那么任何人都能推送，万一群众里有坏人呢？远在印度的阿三给你代码里推送一个木马
    - 推送之前需要更新，不然可能冲突
- 其他开发人员想查看你的代码，拉取 ==Pull==
    - 可能冲突，合并代码
