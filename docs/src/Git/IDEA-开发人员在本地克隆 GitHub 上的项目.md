# 开发人员在本地克隆 GitHub 上的项目

## 克隆

鉴于有同学可能不太熟悉各种情况，罗列以下几种

### 从版本控制创建项目，【File】→【New】→【Project from Version Control】

![image-20210902151236539](https://attach.blog.wen7.online/image-20210902151236539.png)

### 如果当前项目还没加入版本控制，那么导航栏是 【VCS】→【Get from Version Control】

![image-20210902150817306](https://attach.blog.wen7.online/image-20210902150817306.png)

### 如果当前项目已经加入版本控制，【Git】→【Clone】

![image-20210902150919571](https://attach.blog.wen7.online/image-20210902150919571.png)
