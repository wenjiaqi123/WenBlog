# 数据类型

## 基本数据类型

### number

- 数值（正负数，整数，小数）
- Infinity
- -Infinity
- NaN
	- not a number 不是一个数字



### string

- 字符串

	- "hello"
	- "a"
	- 'hello'
	- 没有字符的概念，不管单引号还是双引号，都是字符串

	

### boolean

- true

- false

	

### undefined

- undefined
	- 未定义，变量没有赋值就直接使用就是 undefined

### null

- null
