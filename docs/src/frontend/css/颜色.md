# 颜色

```css
span{
    color: red/black/blue/green/....;
}
```

```css
span{
    color: #FF0000;
}
```

```css
span{
    color: rgb(255,0,0);
}
```

## 十六进制（大小写无关）

```css
color:#FF0000;  // 这种2个2个2个重复的，可以简写 color:#F00;
color:#336699;  // 可以简写 color:#369;
color:#0f0a0b;  //这种就不可简写
```



## RGB颜色（大小写无关）

```css
color:RGB(255,0,0);
color:rgb(33,66,99);
```



## RGBA颜色

- RGBA（红，绿，蓝，透明度），透明度取值 0.0~1.0
