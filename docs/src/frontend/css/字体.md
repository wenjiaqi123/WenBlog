# 字体

```css
span{
    font-size:40px;
    font-style: normal;
    font-weight:normal;
    font-family: Arial;
}
```



## font-family 字体名称

- Arial

- 微软雅黑

- 宋体

- ......

    

## font-size 字体大小

- 12px（最小）

- 14px

- 16px（默认）

- ......

    

## font-style 字体样式

- normal 正常。默认
- italic 斜体
- oblique 斜体



## font-weight 字体宽度

- 可用数字 100,200,300,400,500,600,700,800,900
- normal  正常，默认
- bold 加粗，等于 700
- bolder 特粗
- lighter 细体



## color

- 一般设置字体时，设置字体名称，字体代销，字体样式，字体宽度，字体颜色
- 颜色这里是 color，有初学者会写成 font-color
