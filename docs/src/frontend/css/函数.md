# 函数

## max

max(值1，值2，值3，...)

取最大的一个值

```css
div{
    width: max(80%,1200px);
    height: 100px;
       background-color: #eaa2a2;
}
```



## min

min(值1，值2，值3，...)

取最小的一个值

```css
div{
    width: 1200px;
    height: min(20vh,800px);
    background-color: #eaa2a2;
}
```



## calc

```
div {
    width: 200px;
    height: calc(100vh - 80px);
    background-color: #eaa2a2;
}
```
