# IntelliJ IDEA快捷生成 HTML

## 生成 id

输入 ==div#myId== ，按下 tab 键

```html
<div id="myId"></div>
```



## 生成 class

输入 ==div.myclass== ，按下 tab 键

```html
<div class="myClass"></div>
```



## 生成 ul li

输入 ==ul>li*3== ，按下 tab 键

```html
<ul>
    <li></li>
    <li></li>
    <li></li>
</ul>
```



## 顺序数字

- {} 里面的内容是标签内的内容

- $ 顺序数字

输入 ==h$*6== ，按下 tab 键

```html
<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>
```

输入 ==h{$}*6== ，按下 tab 键

```html
<h>1</h>
<h>2</h>
<h>3</h>
<h>4</h>
<h>5</h>
<h>6</h>
```
